# Win32

A Rust Win32 GUI application using the [windows](https://crates.io/crates/windows) crate.

## Features

- Native Win32 window with menu bar
- 4x4 checkbox grid with column headers (A, B, C, D)
- Message box dialog triggered from menu

## Building

```sh
cargo build
```

## Running

```sh
cargo run
```

## Requirements

- Rust (edition 2021)
- Windows OS
